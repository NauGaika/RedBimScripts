шаг 1. Закрыть Ревит.

Для версии Revit 2017

шаг 2. 
Открыть папку \\picompany.ru\pikp\Dep\LKP4\WORK\scripts\install выбрать все элементы и скопировать (Ctrl+C).

шаг 3. Открыть "Мой компьютер", в адресную строку скопировать и нажать Enter: 
	%programdata%\Autodesk\Revit\Addins\2017



шаг 4. Перейти в папку из шага 2 и вставить через (Ctrl+V). 
При необходимости выбрать "заменить" (если уже ранее было установлено).

шаг 5. Запустить Ревит. Выбрать "Всегда загружать".





Для версии Revit 2019

шаг 2. 
Открыть папку \\picompany.ru\pikp\Dep\LKP4\WORK\scripts\install выбрать все элементы и скопировать (Ctrl+C).

шаг 3. Открыть "Мой компьютер", в адресную строку скопировать и нажать Enter: 
	%programdata%\Autodesk\Revit\Addins\2019



шаг 4. Перейти в папку из шага 2 и вставить через скопированное на предыдущем шаге (Ctrl+V). 
При необходимости выбрать "заменить" (если уже ранее было установлено).

шаг 5. Запустить Ревит. Выбрать "Всегда загружать".
